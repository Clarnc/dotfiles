## Installation
1.rm -rf dotfiles

2.git clone https://github.com/Clarnc/dotfiles

3.cd dotfiles

4.cp -R ./.config/* ~/.config/

5.cp -R ./Scripts ~/

6.manage wallpaper in ~/.config/bspwm/bspwmrc

7.restart your display manager
